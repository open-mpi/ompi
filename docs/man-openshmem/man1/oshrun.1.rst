oshrun
======

.. include_body

oshrun |mdash| Execute serial and OpenSHMEM parallel jobs.

The ``oshrun(1)`` command is functionality identical to the
:ref:`mpirun(1) <man1-mpirun>` command.  See the :ref:`mpirun(1)
<man1-mpirun>` man page for details.
