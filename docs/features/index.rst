Open MPI-specific features
==========================

Open MPI has multiple features that are above and beyond what is
specified in the `MPI Standard <https://www.mpi-forum.org/docs/>`_.

This section does not list *all* Open MPI features that are not
specified in the official MPI Standard, but does list two major
categories of Open MPI-specific features.
