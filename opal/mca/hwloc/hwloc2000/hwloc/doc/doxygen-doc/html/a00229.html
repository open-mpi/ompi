◆ hwloc_pcidisc_tree_attach()

int hwloc_pcidisc_tree_attach ( struct hwloc_topology *  topology,
struct hwloc_obj *  tree 
)

Add some hostbridges on top of the given tree of PCI objects and attach them to the topology.

For now, they will be attached to the root object. The core will move them to their actual PCI locality using hwloc_pci_belowroot_apply_locality() at the end of the discovery.

In the meantime, other backends lookup PCI objects or localities (for instance to attach OS devices) by using hwloc_pcidisc_find_by_busid() or hwloc_pcidisc_find_busid_parent().
